## 修正一：個股數據抓取指令（替換原 STEP 2）

```
### STEP 2｜台股個股數據

對以下每一支個股，依照順序嘗試以下來源，取得昨日（YYYY/MM/DD）收盤數據：

**來源優先順序（依序嘗試，取得後停止）：**
1. 台灣證券交易所官網：https://www.twse.com.tw/zh/trading/historical/stock-day.html
   → 輸入股票代號，查詢最新收盤價與漲跌幅
2. 鉅亨網個股頁面：https://www.cnyes.com/twstock/[股票代號]
   → 查看「即時報價」或「歷史股價」
3. Yahoo Finance 台股：https://tw.finance.yahoo.com/quote/[股票代號].TW

**嚴格規則：**
- 股價必須是「昨日收盤價」，不接受日中價、52週高點、或任何非昨收的數字
- 若無法確認是昨日收盤價，一律標記「待確認」，不得填入任何其他數字
- 籌碼數據從鉅亨網「籌碼面」頁面取得：https://www.cnyes.com/twstock/[股票代號]/chips
- 月營收從公開資訊觀測站取得：https://mops.twse.com.tw/mops/web/t05st10_ifrs

**🚫 嚴禁行為（股價取得）：**
- 嚴禁用 WebSearch 查個股收盤——WebSearch 返回值不可信，曾導致系統性錯誤
- 嚴禁從新聞標題或漲停消息「推算」收盤價（例如：前收 × 1.1）
- 正確做法：用 Chrome 開啟 https://finance.yahoo.com/quote/[代號].TW/history/ 或 .TWO/history/，讀取表格第一列的 Close 欄位
- 若 Chrome 無法取得，明確標記「待確認」，絕不猜測

**個股清單：**
3037 欣興（.TW） / 3189 景碩（.TW） / 8046 南電（.TW）
3491 昇達科（.TWO） / 2313 華通（.TW） / 2367 燿華（.TW）
3163 波若威（.TWO） / 4979 華星光（.TWO）
1519 華城（.TW） / 1503 士電（.TW） / 1513 中興電（.TW）

⚠️ 注意：波若威代號為 3163（上櫃 .TWO），不是 6856。
6856.TWO 是另一家公司（SHINE TREND），請勿混淆。
```

## 修正二：新聞日期嚴格過濾（替換原 STEP 3 前置說明）

```
### STEP 3 前置規則｜新聞時效性篩選

**只納入「過去 48 小時以內」發布的新聞。**

判斷方式：
- 新聞頁面上的發布日期或時間戳記，必須是 [執行日期] 或 [前一天] 的日期
- 若新聞日期不明確，不得納入
- 若某板塊在過去 48 小時內確實無重大新聞，請明確標記：
  「今日無 48 小時內新聞，以下為本週最新動態（日期：YYYY/MM/DD）：」
  然後才列較舊但仍相關的新聞，並標清楚日期

**禁止行為：**
- 不得將日期超過 7 天的新聞列入「今日重點新聞」區塊
- 不得因找不到新新聞而使用模板示意文字
```

## 修正三：強制 HTML 輸出（替換原 STEP 5）

```
### STEP 5｜產出 HTML 報告

**輸出格式：HTML 檔案，副檔名 .html**

執行步驟：
1. 打開桌面「產業晨報」資料夾中的「每日產業晨報_模板.html」
2. 閱讀該檔案的完整 HTML 結構
3. 找到所有以 [佔位符] 標記的欄位
4. 將 STEP 1-4 搜集到的真實數據逐一填入，替換所有佔位符
5. 將修改後的完整 HTML 存為新檔案：「晨報_YYYYMMDD.html」
6. 存入同一資料夾

**重要規則：**
- 不得輸出 Markdown (.md) 格式
- 不得改變模板的 HTML 結構與 CSS 樣式
- 所有標記「待確認」的欄位，在 HTML 中用此格式呈現：
  <span style="color:#b8872a">待確認</span>
- 若某板塊今日無 48 小時內新聞，在對應新聞區塊填入：
  <div class="news-entry"><div class="news-headline">今日無最新動態</div>
  <div class="news-meta">最近相關報導見下方產業解讀</div></div>
```

## 補充：數據品質自我檢查清單

在輸出 HTML 之前，執行以下檢查：

```
□ 每支個股的股價是否標記了資料日期？（格式：NT$XXX，YYYY/MM/DD 收盤）
□ 無法取得的欄位是否標記「待確認」而非填入猜測值？
□ 所有列入「今日重點新聞」的新聞是否在 48 小時內？
□ 新聞來源是否標明媒體名稱與日期？
□ 輸出是否為 .html 格式？
□ 模板的版面結構是否完整保留？
```
